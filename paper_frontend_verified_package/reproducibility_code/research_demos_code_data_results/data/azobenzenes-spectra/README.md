# Azobenzene IR/Raman Data

`IR_Raman_azo.csv` is the experimental fine-tuning dataset released by the authors of:

> Unlocking azobenzene isomerization mechanisms via an LLM agent-driven workflow
> integrating simulation, experiment, and machine learning.

- Paper file: `Paper/d5sc08794e.pdf`
- Source repository: `https://github.com/pic-ai-robotic-chemistry/azobenzenes-spectra`
- Source path: `data/IR_Raman_azo.csv`
- License: MIT, preserved in `LICENSE`
- SHA256: `AD69D8AA4D998B31FC741B1939DDCE4A90CB38AA09641C2E2CBA348A762C9158`

The file contains 20 paired IR/Raman spectra. The first 8,000 columns are spectral
features and column 8,001 is the dihedral-angle target used by the released
fine-tuning code.
