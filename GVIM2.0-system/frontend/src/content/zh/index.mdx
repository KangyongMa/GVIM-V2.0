# GVIM 文档

GVIM 是一个专门面向化学与材料科学领域的 AI Agent 平台。它提供了一个科学计算运行时 Harness，集成了 RDKit 分子分析、pymatgen 材料结构计算、Materials Project 数据检索、Ketcher 化学画板以及安全的沙箱代码执行环境；同时还提供了一个美观易用的 Web 应用，帮助科研人员进行高效的化学结构和材料数据分析与协作。

本文档围绕这两个部分组织：

- **GVIM Harness**：用于构建科学计算 Agent 系统的核心 SDK 和运行时层。
- **GVIM 应用**：构建在 Harness 之上的参考应用，用于化学与材料科学研究工作流、实验数据分析和科研协作。

如果你想了解 GVIM 的工作原理，从简介开始阅读。如果你想基于核心运行时进行开发，请查阅 Harness 文档。如果你想将 GVIM 作为应用部署和使用，请查阅应用文档。
